# BLESerial3
BLESerial3をiOSアプリから使うためのサンプルコードです。
[BLEモジュールBLESerial３の紹介 - 浅草ギ研](http://www.robotsfx.com/robot/BLESerial3.html)

## フォルダ構成

- Arduino: Arduino Micro向けのコードが入っています
- BLESerial3-sample: iOSのコードが入っています
